# RTS Teklo — Equipment Portal

Корпоративный портал для управления оборудованием европейских производителей (Bottero, IRIS Inspection Machines, Fermac, Carmet) на рынке России и СНГ.

## 🔐 Доступ

- **Логин:** `ritsteklo`
- **Пароль:** `2026`

## 📊 Возможности

### 1. Страница входа
- Анимированная форма авторизации
- Защита паролем (статичный)
- Сохранение сессии

### 2. Выбор компании
- Карточки с анимацией hover
- 4 компании: Bottero, IRIS, Fermac, Carmet
- Подготовлено под расширение

### 3. Дашборд IRIS (приоритетный)
- **Статистика:** Всего машин, на гарантии, гарантия истекла, количество заводов
- **Распределение по заводам:** Горизонтальные прогресс-бары
- **Таймлайн:** История поставок по годам
- **Фильтры:** По типу оборудования (Evolution 5, Evolution 16, EasySet, Evolution 1)
- **Карточки оборудования:**
  - Название машины и серийный номер
  - Завод и линия
  - Дата поставки и установки
  - **Визуальная шкала гарантии** с процентом остатка
  - Список запасных частей
  - Комментарии

## 🚀 Размещение на GitHub Pages

### Шаг 1: Создайте репозиторий на GitHub

1. Зайдите на [github.com](https://github.com) и войдите в аккаунт
2. Нажмите зелёную кнопку **"New"** (или "+") → **"New repository"**
3. В поле **Repository name** введите: `rts-teklo-portal`
4. Выберите **Public** (публичный репозиторий)
5. Нажмите **"Create repository"**

### Шаг 2: Загрузите файлы

**Способ A — через веб-интерфейс (проще):**

1. На странице нового репозитория нажмите **"uploading an existing file"**
2. Перетащите файл `index.html` в область загрузки
3. В поле "Commit changes" напишите: `Initial commit`
4. Нажмите **"Commit changes"**

**Способ B — через Git (для опытных):**

```bash
git clone https://github.com/ВАШ_НИК/rts-teklo-portal.git
cd rts-teklo-portal
cp /путь/к/index.html .
git add index.html
git commit -m "Initial commit"
git push origin main
```

### Шаг 3: Включите GitHub Pages

1. В репозитории перейдите в **Settings** (вкладка сверху)
2. В левом меню выберите **Pages**
3. В разделе **"Build and deployment"** → **Source** выберите:
   - **Deploy from a branch**
   - Ветка: `main` (или `master`)
   - Папка: `/ (root)`
4. Нажмите **Save**
5. Подождите 1-2 минуты — появится зелёная плашка с ссылкой вида:
   `https://ваш-ник.github.io/rts-teklo-portal/`

### Шаг 4: Поделитесь ссылкой

Скопируйте ссылку и отправьте начальникам! 🎉

## 📝 Обновление данных

Данные оборудования хранятся в JavaScript-массиве `irisData` внутри `index.html`.

Чтобы обновить данные:
1. Откройте `index.html` в текстовом редакторе
2. Найдите строку `const irisData = [...]`
3. Отредактируйте JSON-массив
4. Сохраните и загрузите на GitHub заново

## 🔒 Безопасность

⚠️ **Важно:** Это клиентская авторизация (JavaScript). Пароль виден в исходном коде. Для внутреннего корпоративного использования это приемлемо, но для публичного доступа рекомендуется добавить серверную авторизацию.
