## 🔄 Как работает синхронизация

**Принцип:** Все данные хранятся в одном JSON-файле (`data/machines.json`) на GitHub. Все пользователи загружают этот файл при открытии портала.

```
Ваш компьютер → Редактор → Скачать JSON → Загрузить на GitHub → Все видят изменения
```

## ✏️ Как редактировать данные (шаг за шагом)

### Шаг 1: Откройте редактор
1. Скачайте файл `editor.html` на свой компьютер
2. Откройте его в браузере (двойной клик)
3. Это ваш личный инструмент редактирования — он работает локально

### Шаг 2: Загрузите текущие данные
1. В редакторе нажмите **"Загрузить JSON"**
2. Выберите файл `data/machines.json` из вашего репозитория на GitHub
3. Данные загрузятся в таблицу

### Шаг 3: Редактируйте
- **Добавить машину:** Нажмите **"+ Добавить новую машину"**, заполните форму, нажмите **"Сохранить"**
- **Редактировать:** Нажмите **✎** на строке машины, измените данные, нажмите **"Сохранить"**
- **Удалить:** Нажмите **×** на строке машины, подтвердите удаление

### Шаг 4: Скачайте обновлённый JSON
1. Нажмите **"Скачать JSON"**
2. Файл `machines.json` скачается на ваш компьютер

### Шаг 5: Загрузите на GitHub
1. Зайдите в свой репозиторий на GitHub
2. Перейдите в папку `data/`
3. Нажмите **"Upload files"** → выберите скачанный `machines.json` → замените старый
4. Commit message: `Обновление данных о машинах`
5. **Commit changes**

### ✅ Готово!
Все пользователи, зайдя по ссылке, увидят обновлённые данные!

## 🚀 Первоначальное размещение на GitHub Pages

### 1. Создайте репозиторий
1. [github.com](https://github.com) → New → New repository
2. Название: `rts-teklo-portal`
3. **Public** → Create repository

### 2. Загрузите все файлы
1. На странице репозитория нажмите **"uploading an existing file"**
2. Перетащите ВСЕ файлы: `index.html`, `editor.html`, и папку `data/`
3. Commit message: `Initial commit v3.0`
4. **Commit changes**

### 3. Включите GitHub Pages
1. **Settings** → **Pages** (в левом меню)
2. Source → **Deploy from a branch**
3. Branch: `main`, Folder: `/ (root)`
4. **Save**
5. Через 1–2 минуты: `https://ваш-ник.github.io/rts-teklo-portal/`
