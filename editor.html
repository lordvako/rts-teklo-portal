RTS TEKLO — Редактор данных

📋 Как редактировать данные:

  1. Нажмите "Загрузить JSON" и выберите файл machines.json из вашего репозитория
  2. Редактируйте машины (✎) или добавьте новую (+)
  3. Нажмите "Скачать JSON" — получите обновлённый файл
  4. Загрузите этот файл на GitHub в корень репозитория (рядом с index.html)
  5. Все пользователи увидят изменения при следующем заходе!
